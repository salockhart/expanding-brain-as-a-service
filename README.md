# API

## `GET /brain`

### Description
Takes in a set of four phrases, for each of the four stages of brain activity.

### Parameters
* first

Brain #1 sentence

* second

Brain #2 sentence

* third

Brain #3 sentence

* fourth

Brain #4 sentence

### Return Format
An image

### Example
**Request**
    
    https://expanding-brain-as-a-service.herokuapp.com/brain?first=who&second=whom&third=whom%27st&fourth=whom%27st%27d
